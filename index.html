CSS3 gallery is built using CSS3 and JavaScript. All animations are done by CSS3. JavaScript is used just to manipulating classes, setting values for animations and handling CSS3 animation events.

Tested on Firefox 5, Chrome 12 and Safari 5.0.5.

The photos are taken in a railway sation located in Glogow/Poland. There is a strange both-side windowed hall which is a passageway to platforms. The gallery presents a collage of three photos of the hall's windows divided into pieces.
